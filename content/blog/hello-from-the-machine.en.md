# Hello from the machine

This is a private starter post for checking the blog locally. Replace it with something real, add a `date`, and remove `draft: true` when it is ready to publish.

## One file is enough

The filename supplies the slug and language. The first heading becomes the title, and the first paragraph becomes the summary shown in the archive.

:::note
Custom blocks stay readable in the Markdown source and become designed elements on the website.
:::

:::details{title="What gets generated automatically?"}
Reading time, heading links, translation pairing, article metadata, RSS entries, and sitemap entries all come from this file.
:::

:::warning
A draft without a publication date never enters the production blog, sitemap, or RSS feed.
:::
