# Привет из машины

Это приватная стартовая запись для локальной проверки блога. Замени её настоящим текстом, добавь `date` и убери `draft: true`, когда запись будет готова к публикации.

## Достаточно одного файла

Имя файла задаёт slug и язык. Первый заголовок становится названием, а первый абзац — описанием в архиве.

:::note
Кастомные блоки остаются читаемыми в Markdown и превращаются в оформленные элементы на сайте.
:::

:::details{title="Что создаётся автоматически?"}
Время чтения, ссылки на разделы, связь переводов, метаданные статьи, RSS и sitemap формируются из этого файла.
:::

:::warning
Черновик без даты публикации никогда не попадёт в production-блог, sitemap или RSS.
:::

## Блоки кода знают свой язык

Используйте обычное обозначение языка после открывающего fence. Подсветка происходит во время рендера страницы, а браузер получает уже готовую разметку.

```ts
const post = "content/blog/hello-from-the-machine.ru.md";
```
